# Simple Android Calculator

A simple calculator app built with Kotlin and Android.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Percentage calculation
- Backspace functionality
- Clean Material Design UI

## Download

Download the latest APK from [Releases](https://github.com/DALI951/android-calculator/releases)

## Build

To build the APK locally:

```bash
./gradlew assembleRelease
```

The APK will be at `app/build/outputs/apk/release/app-release-unsigned.apk`
